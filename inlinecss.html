cascading style sheets (Css)

Css is used to style and describe the layout of a web page

Css can be added in a web page

  1. Inline css
  2. Internal css
  3. External css

Css has some of the following functions in aweb page

  • changing the color of elements
  • adjusting the size and dimensions
  • adjusting the layout of the elements
